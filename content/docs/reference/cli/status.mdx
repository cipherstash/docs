Displays implementation status

```bash
npx stash status [flags]
```

### Flags

| Flag | Description |
| --- | --- |
| `--quest` | Force the quest-log output (emoji + progress bars) even in non-TTY contexts. Default is auto: fancy in a terminal, plain in CI / pipes / agents. |
| `--plain` | Force the plain-text output even in TTY contexts. |
| `--json` | Emit a structured JSON document instead. |
